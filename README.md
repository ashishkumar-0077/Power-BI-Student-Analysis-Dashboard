# 🎓 Student Performance Analysis Dashboard | Power BI



## 📌 Project Overview

The **Student Performance Analysis Dashboard** is an interactive Power BI dashboard built to analyze student academic performance across departments, grades, and result categories. The dashboard provides actionable insights through interactive visualizations and KPIs, helping users monitor performance trends and make data-driven decisions.

---

## 🚀 Dashboard Features

- 📊 Interactive Power BI Dashboard
- 🎯 Dynamic Department Filter
- 📈 Student Distribution by Grade
- 📉 Pass, Average & Fail Analysis
- 📚 Grade-wise Student Count
- 👨‍🎓 Parent Education vs Student Performance
- 📌 Performance & Engagement Analysis
- 📋 KPI Cards for Quick Insights

---

## 📊 Key Metrics

- Overall Average Score
- Student Grade
- Pass Percentage
- Total Students

---

## 📈 Dashboard Insights

- Analyze student performance across multiple departments.
- Compare Pass, Average, and Fail categories.
- Visualize grade distribution from Grade A to Grade E.
- Understand the relationship between parents' education and student outcomes.
- Identify performance trends using interactive charts and filters.
